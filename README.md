# LetsChat-MERN
<h1> MERN Chatting APP</h1>
<h2>Tech Used</h2>
<ul>
<li> React.js  </li>
<li> Node.js  </li>
<li>Express.js  </li>
<li> MongoDb  </li>
<li> Firebase(For Google Sign in)  </li>
<li> Mongoose  </li>
<li> Pusher(To make Mongodb work realtime)  </li>
<li> Material Ui  </li>
</ul>
